# Spring AI RAG Assistant

A simple **Retrieval-Augmented Generation (RAG)** application built with **Spring Boot** and **Spring AI**.

The project allows an AI assistant to answer questions based on your own documents by combining:
- Document processing
- Vector embeddings
- Semantic search
- Large Language Models (LLM)

## Features

- Spring Boot backend
- Spring AI integration
- PDF document loading
- Vector similarity search
- PostgreSQL + pgvector support
- AI-powered question answering
